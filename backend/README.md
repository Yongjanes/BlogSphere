# BlogSphere

- BlogSphere is a full-stack blogging platform where users can create, publish, and interact with blog posts.

- Users can write blogs using Markdown, add images, and publish them with clean, custom URLs. Anyone on the platform can browse blogs written by other users, like posts, and leave comments. Each blog keeps track of engagement such as views, likes, and comments.

- The platform also includes basic admin support for managing users and moderating content, ensuring a safe and organized blogging environment.

- BlogSphere is being built with scalability and backend engineering best practices in mind, focusing on clean data models, analytics, and efficient content handling.
